# Smart Expense Tracker

A responsive React.js web app to help users track their income and expenses, visualize spending with interactive doughnut charts, and maintain a persistent transaction history using the Context API and localStorage.

---

## Features

- Add, delete, and categorize income & expense transactions  
- Real-time data visualization using Chart.js (Doughnut chart)  
- Data persistence using localStorage  
- Global state management with Context API + useReducer  
- Fully responsive layout using Material UI  
- Custom Snackbar notifications for user actions  

---

## Tech Stack

- **React.js** — SPA architecture with functional components  
- **Context API + useReducer** — lightweight global state management  
- **Material UI** — for UI components and styling  
- **Chart.js via react-chartjs-2** — for interactive charts   
- **LocalStorage** — for offline data persistence  

---

## Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/smart-expense-tracker.git

# 2. Navigate into the project directory
cd smart-expense-tracker

# 3. Install dependencies
npm install

# 4. Start the app
npm start

```
